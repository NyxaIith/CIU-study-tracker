# What is an Operating System

## Overview

An operating system (OS) is software that manages a computer's hardware and provides a foundation for programs to run. It sits between applications and hardware, translating high-level requests into physical operations.

Examples: Linux, Windows, macOS, Android.

## Key Points

- The OS manages the CPU, memory, storage, and input/output devices.
- It allows multiple programs to run at once by sharing CPU time (multitasking).
- It provides a file system so programs can store and retrieve data.
- Programs communicate with hardware through the OS via system calls — not directly.
- The **kernel** is the core of the OS. It runs with full hardware access.

## Example

When you save a file, your program does not write to the disk directly. It makes a system call. The OS handles the physical write, manages permissions, and updates the file system.
