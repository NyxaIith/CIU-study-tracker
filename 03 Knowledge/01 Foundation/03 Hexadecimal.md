# Hexadecimal

## Overview

Hexadecimal is a base-16 number system. It uses the digits 0–9 and the letters A–F, where A=10, B=11, C=12, D=13, E=14, F=15.

It exists in computing because one hex digit represents exactly 4 bits, making binary far easier for humans to read and write. The value `11111111` in binary is `FF` in hex.

## Key Points

- Base-16: 16 possible values per digit (0 through F).
- One hex digit = 4 bits. Two hex digits = 1 byte.
- Hex values are often prefixed with `0x` in code (e.g. `0xFF`).
- Colors in web design use hex: `#FF5733` means R=255, G=87, B=51.

## Example

Binary `10110100` → split into groups of 4 → `1011` `0100` → hex `B4`

Decimal 255 → binary `11111111` → hex `FF`
