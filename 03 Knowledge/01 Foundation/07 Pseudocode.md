# Pseudocode

## Overview

Pseudocode is a way of writing algorithms using plain, structured language that resembles code but follows no strict syntax. It is used to plan and communicate logic without worrying about a specific programming language.

## Key Points

- Pseudocode has no standard format — any clear, consistent style works.
- It uses English words for logic: IF, THEN, ELSE, WHILE, FOR, RETURN.
- Indentation shows structure, just as in real code.
- The goal is clarity, not execution.

## Example

Finding the maximum value in a list:

```
function findMax(list):
    max = list[0]
    for each item in list:
        if item > max:
            max = item
    return max
```
