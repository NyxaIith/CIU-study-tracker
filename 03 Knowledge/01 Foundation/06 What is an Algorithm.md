# What is an Algorithm

## Overview

An algorithm is a precise, ordered set of steps for solving a problem. Given the same input, it must always produce the same correct output. It must eventually stop.

The word comes from the name of the 9th-century mathematician Muhammad ibn Musa al-Khwarizmi.

## Key Points

- An algorithm must be **finite** — it must terminate.
- An algorithm must be **unambiguous** — each step has exactly one meaning.
- An algorithm must be **correct** — it produces the right output for all valid inputs.
- Algorithms are language-independent. The same algorithm works in Python, C, or any other language.

## Example

Finding the largest number in a list:
1. Assume the first number is the largest.
2. Compare each remaining number to the current largest.
3. If it is larger, it becomes the new largest.
4. After all numbers are checked, return the largest.
